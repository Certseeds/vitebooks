# 尾声

> 从恩赐中坠落……

燃烧. 不断地燃烧.

我被炎热和自己烧焦的肉体的恶臭惊醒. 我的身体被火焰包围着. 我不需要看就能知道, 我的每根神经末梢都在尖叫.

下降.

我想我已经落入我兄弟的另一个死亡陷阱, 一个火坑或深渊.

但我下降的时间太长, 也太失重了, 不可能是这样. 我睁开眼睛, 在玻璃状的体液沸腾, 然后在眼窝里蒸发之前的几秒钟里, 我看见下面有一个巨大的球体, 透过灼热的烟雾.

这是一个灰色的——几乎是苍白的世界——被白云缭绕着. 我在它上面很远的地方, 没有船, 甚至没有盔甲的保护, 就突破了它的上层大气.

皮肤烧坏了. 还有肉体, 然后是肌肉.

我的头向后扭, 嘴巴张得大大的, 发出无声的尖叫, 我经历了无法估量的痛苦.

星星和星云在我面前闪烁, 但我没有能力看到它们.

当我的大脑反抗我的身体告诉它的时候, 我通过我的心灵之眼目睹了我自己的毁灭.

--------

伏尔甘, 他的身体是地狱…

皮肤像羊皮纸一样皱缩, 血液像唾沫四溅

他的肉在剥落和分解

伏尔甘, 被渲染成黑色的骨头.

他干瘪的骨架冲破了上层大气, 直到最后……

伏尔甘死去了.

--------

完
