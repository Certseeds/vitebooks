# CLAUDE.md

本文件为 Claude Code (claude.ai/code) 在此仓库工作时提供指导。

## 内容结构

每个书籍目录包含:

- `organize.md` - 描述书记内容, 属于元数据
- `meta.md` 包括来源, 以及进行的编辑, 属于元数据.
- `base.md` - 部分书籍使用此模式存放单文件内容, 内部可能蕴含出场人物表
- `src/chapter*.md` - 章节内容文件
- `names.txt`内部是人名替换表, 每一行左侧的人名需要被替换成右侧的, 配合 `base.md`
- 忽略 `allnames.txt`, 这是脚本提取的所有人名表, 带有大量错误.
- `meta.toml` toml格式元数据, 内部包括书籍序号, 作者, 阵营关键字等数据.

## 用户指令

1. 所有对话都用中文进行
2. 本仓库中使用全英文标点
3. 需要使用朴素的方式, 一本一本书, 一个文件接着一个文件, 一个段落接另一个段落的对每一个 md 文件进行 review
4. 基础的任务以书为一个单位, 执行结束后ask Question to user

### 子任务-检查中文语法

> 可以参考 base.md 中的出场人物表, 以及 names.txt 中的人名映射表进行人名替换.

1. 目的是检查中文基础语法, 拼写错误, 标点错误等基础错误, 发现后直接修改无需询问
2. 只能进行字, 词级别的修改
3. 尽全力edit, 如果失败, 则输出错误的一行并继续, 不再钻牛角尖
