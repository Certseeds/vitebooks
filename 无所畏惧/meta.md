# 无所畏惧-元数据

+ 作者: 张拙言
+ 出处: <https://www.bilibili.com/read/cv2222305/>

+ 原译者: haldir_
+ 原帖: <http://tieba.baidu.com/p/2960726169>

## 阅读顺序

0. 长篇《无所畏惧》、漫画《马库拉格之耀》
1. 长篇《背叛者》(节选)
2. 超短篇《纪念厄努阿利乌斯》
3. 超短篇《征服者之心》
4. 长篇《不被铭记的帝国》
5. 长篇《Pharos》(节选)

> 这一系列故事围绕勇敢而理想主义的极限战士军团展开, 叛军将极限战士的家园"五百世界"化为血腥战场与莫测的政治舞台, 势单力薄的极限战士怎样才能力挽狂澜?
